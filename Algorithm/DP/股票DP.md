# 股票DP

prices[i] 表示第 i 天股票的价格，求在给定限制条件下的最大收益

dp[i][k] 表示第 i 天的状态（持有，非持有，卖出，冷冻，第几次操作）
### 限制购买次数
限制购买次数为 k 次，并且手头最多同时存在一支股票
dp[i][2 * k - 1] 表示第 i 天持有第 k 支股票的最大现金剩余（前一天也持有/前一天不持有第 k - 1 支今日买入），dp[i][2 * k]表示第 i 天不持有第 k 支股票的最大现金剩余（前一天不持有/前一天持有第 k 支今日卖出）


```cpp
for (int i = 1; i < 2 * k; i += 2) {
        dp[0][i] = -prices[0];
    }

    for (int i = 1; i < prices.size(); ++i) {
        for (int j = 1; j < 2 * k; j += 2) {
            dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - 1] - prices[i]);
            dp[i][j + 1] = max(dp[i - 1][j + 1], dp[i - 1][j] + prices[i]);
        }
    }
```
### 卖出后限制冷冻天数
每天的状态分为：持有、卖出、非卖出的非持有、冷冻，把非持有分成有无卖出是为了便于冷冻的转移
