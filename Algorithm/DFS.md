# DFS
深度优先搜索，使用递归函数实现暴力枚举

从初始状态出发，选择一个可能的“分支”一直深入，直到无法继续，然后回溯到上一个状态，尝试另一个分支

用递归理解：一次搜索可能有多个递归，如果当前元素如果符合条件，就递归到此元素，对于其他可能符合条件元素的递归在此元素之后进行
## DFS 和 栈
递归和栈密不可分，递归本身就是利用函数的栈式调用

DFS一般也可以用栈很好的实现，使用一个 bool 变量表示在栈顶位置是否找到更深的分支，如果找到了就入栈并退出寻找，没有找到就弹出栈顶（回溯）。

除了严格按照深度优先的思路进行，对于栈顶位置，我们也先记录栈顶再弹出（由于所有子路径被探索，栈顶不会被回溯），接着一次性把所有可能路径入栈即可（可以避免重复路径检验）
